# Telegraph Pro

Une interface web moderne pour publier des articles sur Telegraph (telegra.ph) avec :
- Éditeur riche (gras, italique, titres, listes, images)
- Upload d'images via ImgBB
- Token Telegraph pour connexion réelle
- Export/Import de config

## Fonctionnalités
- Édition WYSIWYG simple
- Support images (clé API ImgBB requise)
- Publication directe via API Telegraph
- Sauvegarde locale du token (localStorage)

## Déploiement
Hébergé sur Vercel : [https://telegraph-pro-mod-prod.vercel.app](https://telegraph-pro-mod-prod.vercel.app)  
(Change l'URL une fois que Vercel t'a donné la tienne)

## Comment utiliser
1. Obtiens un token via @telegraph bot sur Telegram
2. Colle-le dans le champ "Token Telegraph"
3. (Optionnel) Ajoute une clé ImgBB pour les images
4. Écris → Publie !

Développé en pur HTML + JS + Tailwind CSS.
